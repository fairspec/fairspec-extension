<h2>Features</h2>

<CardGrid>
  <Card title="Simple" icon="rocket">
    Publish a Fairspec extension in minutes, without the need for coding. Generate Python and TypeScript SDKs automatically.
  </Card>
  <Card title="Modern" icon="document">
    The extension template use the latest version of the Fairspec Standard, allowing to implement basically any business domain logic.
  </Card>
  <Card title="Standard" icon="open-book">
    The extension template is based on the widely-used Fairspec standard, making it compatible with its software ecosystem.
  </Card>
  <Card title="Automatic" icon="laptop">
    The extension template generates and publishes Python and TypeScript SDKs, simplifying the end-user experience of publishing and consuming data packages.
  </Card>
